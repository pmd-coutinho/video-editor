# TanStack Start App

Blank React application built with TanStack Start, TanStack Router, Vite, TypeScript, pnpm, and Tailwind CSS.

## Development

```bash
pnpm install
pnpm dev
```

The development server runs at `http://localhost:3000`.

## Commands

```bash
pnpm generate-routes
pnpm build
pnpm preview
pnpm test
```

## Environment

No environment variables are required for the blank starter. Keep local secrets in `.env` or `.env.local`; both are ignored by Git.
